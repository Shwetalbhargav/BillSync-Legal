# BillSync Legal Frontend

React + Tailwind frontend for BillSync Legal.

## Branch

`feat/frontend-bootstrap`

## Scope

This branch creates the app foundation:

- Vite React app.
- Tailwind theme matching Stitch designs.
- Public auth layout.
- Protected app shell.
- Sidebar and mobile bottom navigation.
- Role-aware navigation preview.
- Route catalog for product screens.
- 24 fallback state routes.
- API service boundary and backend-gap docs.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
