# Responsive Accessibility Copy Polish

This branch improves shared frontend quality without adding new product data contracts.

## Polish Areas

- Visible keyboard focus across links, buttons, form controls, tabs, tables, and shell navigation.
- Reduced-motion support for animations and transitions.
- Safer mobile text wrapping for cards and long record names.
- Keyboard-scrollable data tables with captions.
- Mobile header search access and safer assistant placement above the bottom navigation.
- BillSync background color aligned with the product palette.

## Verification Focus

- Check representative protected and public routes at 360px, tablet, and desktop widths.
- Confirm long text wraps instead of pushing the page sideways.
- Confirm focus is visible while tabbing through shell controls.
- Confirm user-facing copy stays calm and non-technical.
