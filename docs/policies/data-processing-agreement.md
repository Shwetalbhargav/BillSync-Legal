# Data Processing Agreement

Lexora acts as processor for customer workspace data and as controller for limited account, billing, security, and operational telemetry.

Processing purposes:
- Provide the application.
- Secure and monitor the service.
- Support customers.
- Maintain backups and audit trails.
- Comply with law.

Subprocessors must provide appropriate security controls and confidentiality obligations. Production secrets are stored in a managed secret service. Backups are encrypted. Customers may request a subprocessor list before pilot launch.

Data deletion and export follow `docs/policies/data-export-deletion.md`.
